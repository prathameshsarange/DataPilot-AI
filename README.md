# 📖 Overview

DataPilot AI is a production-ready AI application that transforms resumes into structured career intelligence.

The platform combines **Google Gemini AI**, **Pydantic validation**, and a **Streamlit dashboard** to generate professional career reports, detect missing skills, recommend certifications, create learning roadmaps, and provide interview preparation.

It also includes an intelligent Dataset Analyzer capable of understanding CSV datasets and recommending machine learning approaches.
