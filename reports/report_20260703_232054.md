# Learning Roadmap
**Day 30:**
- Master Version Control (Git): Learn fundamental commands (add, commit, push, pull), branching, merging, and collaboration workflows using platforms like GitHub or GitLab.
- Learn React Fundamentals: Understand component-based architecture, JSX, props, state, event handling, and basic routing with React Router.

**Day 60:**
- Integrate TypeScript with React: Grasp TypeScript basics (types, interfaces, enums) and apply them to build type-safe React components, props, and state.
- Explore SQL Databases (e.g., PostgreSQL/MySQL Fundamentals): Learn relational database concepts, basic SQL syntax (SELECT, INSERT, UPDATE, DELETE), table creation, and primary/foreign key relationships.

**Day 90:**
- Implement Unit Testing in Python (Pytest): Learn to write effective unit tests for existing Python applications using Pytest, including fixtures and assertions, and understand test coverage.
- Deepen React Knowledge & State Management: Explore advanced React Hooks (useEffect, useContext, useRef), Context API for state management, and get an introduction to a modern state management library (e.g., Redux Toolkit or Zustand).
- Connect Databases with Python Frameworks: Learn to integrate and interact with SQL databases (PostgreSQL/MySQL) from your existing Django or Flask applications using ORMs (e.g., Django ORM, SQLAlchemy) for CRUD operations.
