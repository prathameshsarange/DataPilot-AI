# 12 Week Learning Roadmap

This roadmap is tailored for a first-year student aiming for Data Science, building upon your existing strengths.

**Phase 1: Foundations & Data Wrangling (Weeks 1-4)**

*   **Week 1-2: Advanced Python & SQL Fundamentals**
    *   **Python:** Deeper dive into OOP, data structures (stacks, queues, trees - conceptual), decorators, generators, error handling.
    *   **SQL:** Learn SQL basics (SELECT, FROM, WHERE, GROUP BY, ORDER BY, HAVING), Joins (INNER, LEFT, RIGHT, FULL), Subqueries, Aggregate Functions. Practice with platforms like LeetCode SQL or HackerRank SQL.
    *   **Project:** Perform basic data extraction and aggregation on a small relational dataset using SQL queries.
*   **Week 3-4: Data Analysis with Pandas, NumPy & Visualization**
    *   **Pandas & NumPy:** Advanced indexing, multi-index, merging, concatenating, `apply`, `groupby` operations, time series data handling.
    *   **Data Visualization:** Matplotlib & Seaborn for static plots (scatter, line, bar, histogram, box plots, heatmaps). Understand choosing the right plot type.
    *   **Project:** Conduct an Exploratory Data Analysis (EDA) on a medium-sized dataset (e.g., Kaggle Titanic dataset), generating insights and visualizations.

**Phase 2: Statistics & Core Machine Learning (Weeks 5-8)**

*   **Week 5-6: Statistics & Probability for Data Science**
    *   **Statistics:** Descriptive statistics (mean, median, mode, variance, std dev), distributions (Normal, Bernoulli, Binomial), Central Limit Theorem.
    *   **Inferential Statistics:** Hypothesis testing (t-tests, ANOVA - conceptual), p-values, confidence intervals.
    *   **Project:** Apply statistical tests to your EDA project to validate hypotheses.
*   **Week 7-8: Intermediate Machine Learning with Scikit-learn**
    *   **Algorithms:** Linear/Logistic Regression (review/deepen), Decision Trees, Random Forests, SVM, K-Nearest Neighbors, Clustering (K-Means).
    *   **Model Evaluation:** Precision, Recall, F1-score, ROC-AUC curve, Confusion Matrix, Cross-validation, Hyperparameter Tuning (GridSearchCV, RandomizedSearchCV).
    *   **Project:** Build and evaluate multiple classification/regression models on a new dataset, comparing their performance using various metrics.

**Phase 3: Deepening & Application (Weeks 9-12)**

*   **Week 9-10: Introduction to Deep Learning OR Cloud Fundamentals**
    *   **Option A (Deep Learning):** Basics of Neural Networks, Keras/TensorFlow (intro to dense layers, activation functions, building a simple ANN for classification).
    *   **Option B (Cloud):** AWS/GCP/Azure fundamentals for data science (S3/Blob Storage, EC2/VMs, basic understanding of managed ML services like SageMaker/AI Platform).
    *   **Project:** If Option A, build a simple deep learning model. If Option B, deploy your previous ML model using Flask on a cloud VM.
*   **Week 11-12: Capstone Project & Portfolio Development**
    *   **Capstone Project:** Choose a more complex dataset or problem. This project should integrate SQL (if applicable), advanced data analysis, multiple ML models, comprehensive evaluation, and compelling visualizations.
    *   **Portfolio:** Document all your projects on your GitHub and update your personal portfolio website with detailed explanations, code, and insights.
    *   **Resume Refinement:** Update your resume with new skills and the capstone project.
